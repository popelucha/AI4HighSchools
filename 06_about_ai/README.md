# Proč se dozvědět o AI víc

Největší výzvy dneška spojené s AI:

- misinformace, dezinformace
- zbraně řízené AI
- ochrana soukromí a duševního vlastnictví
- sociální témata (změna pracovního trhu, změna vzdělávání, ...)
- energie
